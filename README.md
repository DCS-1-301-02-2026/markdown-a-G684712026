


# 情報工学科の紹介
G68400-2026 高木　涼輔
<!-- Markdown記法を使って学科の紹介ページを作る -->

![Takushoku University](hachioji.jpg "八王子国際キャンパス")

八王子国際キャンパスにある情報工学科では、これからの社会で必要とされるコンピュータの知識を基礎から学ぶことができます。緑豊かな落ち着いた環境で、充実した学生生活を送ることができます。

### どんなことを学ぶの？ :+1:
難しい知識がなくても大丈夫です。基礎から順番に学べるカリキュラムになっています。
- パソコンの基本的な使い方
- プログラミングの基礎
- インターネットの仕組みと安全な使い方

**キャンパスの主な施設**
学生が自由に使える便利な施設が揃っています。
1. コンピューター室　**パソコン**を使って実習や課題をやります。
2. 図書館 授業で使う教科書から専門書まで、たくさんの本があります。
